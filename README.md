# Treinando Lógica de Programação com Python

Este repositório é o meu "campo de treinamento" pessoal. Aqui, registro os exercícios de lógica de programação que venho resolvendo para fortalecer minha base e aprimorar minha resolução de problemas em Python.

## 🧠 Meu Método de Aprendizado
Para garantir que eu não esteja apenas "decorando" código, aplico os **4 Pilares do Pensamento Computacional** em cada desafio:

* **Decomposição**: Divido problemas complexos em tarefas menores e gerenciáveis.
* **Reconhecimento de Padrões**: Identifico similaridades entre exercícios para aplicar soluções aprendidas anteriormente.
* **Abstração**: Filtro o que é essencial para resolver o problema, ignorando detalhes desnecessários.
* **Algoritmos**: Escrevo o passo a passo lógico para cada subproblema definido.

## 📂 Organização
* Em breve, organizarei os exercícios por tópicos (ex: sequenciais, condicionais, laços de repetição, etc.).
